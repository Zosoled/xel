Icon

<x-icon> is used to display a specific icon from an SVG iconset file.

Xel comes with several predefined iconsets containing the most often used icons:

  • Material - located in /node_modules/xel/iconsets/material.svg
  • Material Outlined - located in /node_modules/xel/iconsets/material-outlined.svg
  • Fluent - located in /node_modules/xel/iconsets/fluent.svg
  • Fluent Outlined - located in /node_modules/xel/iconsets/fluent-outlined.svg

It is recommended that you create your own iconset that contains only the icons needed by your app. This could be done e.g. by copying a predefined iconset file and then manipulating it with Boxy SVG.
